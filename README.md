# Cautionary Accidentals
A plugin for MuseScore 4 that adds cautionary accidentals to your score.<br>Choose from a wide range of settings, to control when to add them and how they look!

### [Download](https://github.com/XiaoMigros/cautionary-accidentals/archive/main.zip)

## Features
- Add or remove cautionary accidentals from your score
- Choose from a wide variety of settings
- Live updating settings previews, to show you exactly what you get
- Sleek & modern UI styled after MuseScore 4

## Installation
Download the [latest release](https://github.com/XiaoMigros/cautionary-accidentals/archive/main.zip), extract the files and move them to MuseScore's 'Plugins' folder, then follow the standard plugin installation process for your MuseScore version.
[Guide for MuseScore 3](https://musescore.org/handbook/3/plugins#installation) | [Guide for MuseScore 4](https://musescore.org/handbook/4/plugins#install-new)
